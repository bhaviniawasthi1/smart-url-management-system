Shorten. Track. Own Your Links.

LynkForge is a smart URL management platform that turns long, messy links into short, clean, trackable ones. You can password-protect them, set them to auto-expire, and see exactly who clicked them — all from one dashboard.

🚀

URL Shortening

Convert long URLs into short, shareable links using secure Base62 encoding.

🔒

Password Protection

Secure sensitive links behind a BCrypt-hashed password. Only the right people get in.

⏰

Link Expiration

Set expiry dates for time-sensitive links. Auto-expire after 1 hour, 24 hours, or any period.

📊

Click Analytics

Track total clicks, daily/weekly/monthly stats, referrers, and user agents per URL.

🛡️

Admin Controls

Monitor platform activity, manage users, and maintain your URL ecosystem.

⚡

Blazing Fast

Built on Spring Boot 3 with JWT auth, rate limiting, and in-memory caching for speed.
